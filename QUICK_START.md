# 🚀 Quick Start - Get Emails Working in 5 Minutes!

## Step 1: Create Resend Account (2 minutes)
👉 https://resend.com/signup

## Step 2: Get API Key (1 minute)
1. After signup → Dashboard
2. Click "API Keys" → "+ Create API Key"
3. Copy the key (starts with `re_`)

## Step 3: Add API Key (1 minute)
Create a file named `.env.local` in your project root:

```env
RESEND_API_KEY=re_paste_your_key_here
```

## Step 4: Restart Server (30 seconds)
```bash
npm run dev
```

## Step 5: Test! (30 seconds)
Fill out any form on your website and check `evonnchargingsolutions@gmail.com`
